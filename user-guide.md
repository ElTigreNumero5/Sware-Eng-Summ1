# Track My Attendance - User Guide

This guide explains how to use the Track My Attendance app.

The app can be accessed [here](https://eltigrenumero5.github.io/Sware-Eng-Summ1/), and opens on the front page.

## Contents

There are three pages in the app:
- Track my attendance: The front page, shows historic attendance data and current target, and allows navigation to the other pages
- Update my attendance: Functionality to set the current attendance target
- Show my attendance: Display of key information outputs
