# Track My Attendance - User Guide

This guide explains how to use the Track My Attendance app.

The app can be accessed [here](https://eltigrenumero5.github.io/Sware-Eng-Summ1/), and opens on the front page.

## Contents

There are three pages in the app:
- Track my attendance: The front page, shows historic attendance data and current target, and allows navigation to the other pages
- Update my attendance: Functionality to set the current attendance target
- Show my attendance: Display of key information outputs

### Update my attendance

<p align='center'>
    <img src="images\input_page.png" alt="Input page of the Track My Attendance app" width="300">
</p>

An empty container dominates this page. It will later show an interactive calendar where users can change the status of individual days to record attendances and ineligible days.

The main functionality on this page is the *Set target* button. Clicking this button opens a user input box that prompts you to update your attendance target percentage. This must be a whole number 0-100, and reflects your personal attendance target. Updating this value will change results of outputs - you will see it reflected in the *Monthly target* box on this page.
